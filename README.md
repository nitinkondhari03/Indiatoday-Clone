## India today-Clone
India Today is a weekly Indian English-language news magazine published by Living Media India Limited. A website for live news reading and watching the live news

## Tech Stack
 HTML | CSS | JavaScript

 ## Features

 - Sign Up
 
 - Sign In
 
 - Read Newz
 
 - Watch Newz

 - Live Newz Watch
